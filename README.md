## 部署方法 Installation

* 获取源码 Getting the project

```shell
git clone https://github.com/Vivi1838/fileManage.git
```

* web服务器虚拟主机 Setting up the webserver

如果在本地部署，修改本地的hosts记录。

例如windows中，编辑....

Ubuntu中编辑/etc/hosts
```shell
sudo nano /etc/hosts
```
加入
```shell
127.0.0.1     filemanager
```


OSX中编辑/etc/hosts
```shell
sudo vim /etc/hosts
```
加入
```shell
127.0.0.1     filemanager
```

如果是从公网访问，将域名解析到本地主机的IP地址。
